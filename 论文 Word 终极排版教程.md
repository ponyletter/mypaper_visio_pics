# 🎓 硕士学位论文 Word 终极排版教程

## 问题 1：图片排版与子图处理

在学术论文中，单图和多子图的排版逻辑是不同的。利用**“无边框表格”**来固定图片位置是最稳妥的方法，绝对不会乱跑。

### 1. 单张图片的排版
单张图片不需要用表格，直接插入即可：
1. 光标放在独立的新行。
2. 插入图片，将图片的环绕方式设置为**“嵌入型”**（选中图片 -> 右上角布局选项 -> 嵌入型）。
3. 选中图片，点击【开始】选项卡中的**“居中对齐”**。
4. 图片下方按下 `Enter`，插入题注（见问题4）。

### 2. 多张子图组合（如：包含 a, b, c, d 子图）
*以你的论文“图5-5 溯源系统工作界面”为例（包含 a, b, c, d 四个子界面）。*

**实操步骤：**

1. **插入表格**：插入一个 **2行2列** 的表格（如果你有上下两排，每排两张图）。
2. **放入图片**：在第一行的两个单元格中，分别插入图(a)和图(b)，设为“嵌入型”，并让单元格内容**居中对齐**。
3. **编写子图注**：在第二行的两个单元格中，分别输入 `(a) 基地源头种植参数录入` 和 `(b) 产线加工包装批次创建`，并**居中对齐**。
   *（如果有图c和图d，就在下方再增加两行，重复上述操作）*
4. **隐藏表格边框（最关键一步）**：
   * 点击表格左上角的十字箭头 `✥` 选中整个表格。
   * 进入【表格设计】选项卡 -> 【边框】 -> 选择**“无框线”**。
5. **添加总图注**：在整个表格的下方按 `Enter`，插入总图注：`图5-5 溯源系统工作界面`，并居中。

> **💡 小贴士**：选中表格，右键选择“表格属性” -> “选项”，可以将单元格的左右边距调小，让子图靠得更紧凑。

## 问题 2：表格的排版（标准三线表）

学术论文严格要求使用**三线表**（只有顶线、底线和表头线，没有竖线和内部横线）。

**实操步骤：**
1. **插入总标题**：表格的标题必须在表格**上方**（如：`表4-1 密码算法时间开销对比测试结果`），并居中。
2. **清除默认样式**：选中插入的表格，在【表格设计】中选择最普通的“网格表”，然后点击【边框】-> 选择**“无框线”**（此时表格边线变成虚线）。
3. **绘制三线**：
   * **顶线与底线（粗线）**：选中整个表格，点击【边框】旁边的下拉箭头 -> 【边框和底纹】。将线宽设置为 **1.5 磅**。在右侧预览区，点击**上边框**和**下边框**，点击确定。
   * **表头线（细线）**：单独选中表格的**第一行**（表头），再次进入【边框和底纹】。将线宽设置为 **0.75 磅** 或 **1 磅**。在预览区点击**下边框**，确定。
4. **对齐方式**：表头通常居中对齐，数据列根据内容选择居中或左对齐（若为数字，尽量小数点对齐或居中）。

## 问题 3：参考文献的对齐与排版

很多同学的参考文献第二行会顶格，显得非常乱。标准的格式应该是**悬挂缩进**。

**实操步骤：**
1. 将所有参考文献录入完毕，前面加上编号 `[1]`, `[2]`。
2. 选中所有的参考文献文本。
3. 右键选择**“段落”**。
4. 在“缩进”模块中，找到“特殊格式”，选择**“悬挂缩进”**。
5. 磅值通常设置为 **2字符** 或者手动输入 `0.75 厘米`（根据你方括号的宽度微调，直到第二行文字和第一行文字对齐，而不是和方括号对齐）。
6. 行距一般设置为“单倍行距”或“1.25倍行距”。

## 问题 4：交叉引用（图、表、公式的自动更新）

如果你在正文中手打“如图5-1所示”，一旦前面加了一张图，后面的编号全都要手动改，非常崩溃。必须使用**“题注 + 交叉引用”**。

### 第 1 步：给图表生成自动编号（题注）
1. 选中你的图片（或表格）。
2. 点击上方菜单栏【引用】 -> **【插入题注】**。
3. 点击“新建标签”，输入“图”（或者“表”、“公式”）。
4. 勾选“包含章节号”（需要你的标题应用了多级列表），它会自动生成“图5-1”。在后面补上文字“溯源系统网络启动界面”。

### 第 2 步：在正文中交叉引用
1. 在正文打字：“如”
2. 点击【引用】 -> **【交叉引用】**。
3. 引用类型选择“图”。
4. **引用内容选择“只有标签和编号”**（这步极其重要，否则会把图的标题文字全引进来）。
5. 在列表中选择“图5-1”，点击插入。此时正文会变成“如图5-1”。

> **💡 终极绝招**：如果修改了图表顺序，只要按 `Ctrl + A` 全选文档，然后按下键盘上的 **`F9`** 键，所有的图表编号和正文引用会**瞬间自动更新**！

## 问题 5：公式排版（居中与编号靠右）

要求：公式在页面正中间，编号如 `(4-3)` 紧贴页面最右侧。

### 推荐方法：Word 自带的终极魔法（最快最好用）
如果你使用的是 Office 2016 及以上版本或 WPS 最新版：
1. 新起一行，按下 `Alt + =` 快捷键，插入一个公式框。
2. 在公式框内输入你的公式内容（例如 `r=e+x_1 n`）。
3. 在公式代码的最末尾（但在公式框内部），输入一个井号 `#`，然后紧接着输入你的编号 `(4-3)`。
   * 看起来像这样：`r=e+x_1 n#(4-3)`
4. **按下回车键（Enter）**。
5. **奇迹发生**：公式瞬间完美居中，编号 `(4-3)` 瞬间完美靠右！

*(如果你的 Word 版本不支持此快捷键，请使用**问题1**中的隐形表格法：插入1行3列表格，左单元格空白，中单元格放公式并居中，右单元格放编号并右对齐，最后取消表格框线。)*

## 问题 6：目录更新与页码检查

盲审和最终归档时，页码错误是致命伤。硕士论文的页码要求通常是：**摘要/目录使用罗马数字（I, II, III），正文从第1章开始使用阿拉伯数字（1, 2, 3）。**

### 1. 隔离目录与正文（分节符）
1. 将光标放在“目录”的最后一页的最末尾。
2. 点击【布局】 -> 【分隔符】 -> **【下一页（分节符）】**。（注意：是分节符，不是分页符！）
3. 双击“第1章 绪论”这一页的页脚，进入页眉页脚编辑模式。
4. 在上方菜单栏，**取消高亮“链接到前一条文字”**（千万别漏掉，这意味着正文的页脚不再跟随目录的页脚）。

### 2. 设置不同页码
1. 光标停在“目录”页脚，【插入】->【页码】->【设置页码格式】，编号格式选择 `I, II, III...`，起始页码填 `I`。
2. 光标停在“第1章”页脚，【插入】->【页码】->【设置页码格式】，编号格式选择 `1, 2, 3...`，起始页码填 `1`。

### 3. 更新目录
在所有格式排完、图片加完之后：
1. 找到论文前面的目录。
2. 右键点击目录 -> **“更新域”** -> **“更新整个目录”**（不仅更新页码，也更新标题文字）。

## 问题 7：中英文混排字体（一段内设置宋体+Times New Roman）

逐个选中英文字母改字体太痛苦了。Word 原生支持在同一段落内为中英文设置不同字体。

**实操步骤：**
1. 选中你需要调整的段落（或者直接 `Ctrl + A` 全选全文，但避开封面）。
2. 按下快捷键 **`Ctrl + D`**（或右键 -> 字体），打开字体设置对话框。
3. 对话框分为两部分：
   * **中文字体**：在下拉列表中选择 **宋体**。
   * **西文字体**：在下拉列表中选择 **Times New Roman**。
4. 点击“确定”。

> **💡 进阶操作（修改“样式”）**：
> 强烈建议你右键顶部样式栏的**“正文”样式** -> 点击**“修改”** -> 左下角**“格式”** -> **“字体”**。在这里面统一设置中文字体为宋体，西文字体为 Times New Roman。这样你以后打出的每一行“正文”都会自动遵循这个规则，一劳永逸。
